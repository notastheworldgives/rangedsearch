=== Ultimate Member - Range Search ===
Requires at least: WP 4.0.0, PHP 5.3, Ultimate Member 1.3.71
Tested up to: 4.3
Stable tag: 4.3

Adds range search functionality to directory page of Ultimate Member plugin.

== Description ==

This is an addon for Ultimate Member plugin. It makes Ultimate Member plugin to provide range search for selected
searchable fields of numeric type. It also provides the options to set prefix for placeholder of inputs taking lowest
and highest numbers of range.


== Installation ==

1. Upload `mes_um_rangesearch.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
